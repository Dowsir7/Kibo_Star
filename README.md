the idea is that each finger has its own button.  5 buttons can make 32 combos, so that's 26 letters and a bumit of extra functionality like space, enter, etc.

combinations of presses make different letters, so if thumb make a and pointer makes be then thumb and pointer together make c.

bit of a stenographer's tool, I guess.  The idea certainly isn't mine, its been around for the better part of a century.  Just never saw one for Andriod.
